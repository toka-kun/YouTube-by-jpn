# わかめさん談要約

## Educationでの再生について

`Education`での再生は、元々教育機関向けの有償コンテンツです。再生にはキーが必要ですが、これは一時的なもので、通常は3日程度で有効期限が切れます。そのため、再生するたびにキーを更新する必要があります。

### 再生方法

以下のリンクフォーマットを使うことで、エラーが出ずに動画を再生できます。

https://www.youtubeeducation.com/embed/(動画のID)(パラメーター)
